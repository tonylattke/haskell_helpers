# Haskell Helpers

Interactive Console:

ghci

On the interactive console load file use:

:load file_name

Compile and run file:

runhaskell file_name

Run file and interact:

ghci file_name

Compile file:

ghc file_name

Get tipe of the function:

:t function_name
